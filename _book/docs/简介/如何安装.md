## 如何安装

Colly框架安装的先决条件只有一个，那就是golang语言。你可以参考官网文档来安装golang语言，官网文档地址为：https://golang.org/doc/install



在你安装好golang语言之后就可以进行下一步。

### 安装Colly

在终端命令行执行以下命令即可安装Colly框架。

```sh
go get -u github.com/gocolly/colly/...
```
